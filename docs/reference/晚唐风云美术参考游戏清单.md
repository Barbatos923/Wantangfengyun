# 《晚唐风云》美术参考游戏清单

更新时间：2026-04-12
定位：参考清单 / 风格对照表

---

## 1. 使用方式

这份清单不是为了让《晚唐风云》去模仿某一个现成游戏，而是帮助项目建立“组合参考”。

原因很简单：

- 没有任何单一游戏同时满足“CK3 级系统地图 + 中国晚唐气质 + 文书式 UI + 高密度策略信息”这四个目标。
- 如果只参考一个项目，最后很容易要么失去题材辨识度，要么失去策略游戏完成度。

因此建议采用：

- 一组主参考游戏
- 一组辅助参考游戏
- 针对不同模块分别借鉴

---

## 2. 总体建议

建议《晚唐风云》的参考体系如下：

### 主参考

- Crusader Kings III
- Total War: THREE KINGDOMS
- Old World

### 辅参考

- Pentiment
- Oriental Empires

如果要一句话概括：

- 地图系统学 `CK3`
- 中国权力气质学 `全面战争：三国`
- 信息架构与 UI 克制感学 `Old World`
- 材质与时代一致性学 `Pentiment`
- 中国大地图空间气质学 `Oriental Empires`

---

## 3. 主参考游戏

## 3.1 Crusader Kings III

### 适合参考的部分

- 地图层级感
- 政治边界表达
- 地图模式切换逻辑
- 大地图与信息面板的关系
- 顶部资源区和右侧入口区的系统布局
- 角色驱动策略游戏的信息组织方式

### 不适合直接照搬的部分

- 欧式中世纪审美母题
- 欧式纹章语言
- 纯欧洲封建视觉符号
- 角色头像、边框、卷轴等具体样式

### 对《晚唐风云》的启发

CK3 最值得学的不是“长什么样”，而是“怎么组织地图与权力信息”。

对你们而言，它更像：

- 地图机制参考
- 信息结构参考
- 交互层次参考

不是：

- 中国历史美术参考

### 适用模块

- GameMap
- 资源栏布局
- 侧边功能入口
- 人物面板结构
- 地图模式系统
- 战争与疆域边界表达

---

## 3.2 Total War: THREE KINGDOMS

### 适合参考的部分

- 中国历史策略游戏的整体气质
- 军政系统的东方权力感
- 将领、部队、势力的视觉权重
- 战争界面与政治界面的统一性
- 汉字环境下的界面节奏

### 不适合直接照搬的部分

- 过于写实或偏战役演出化的表现
- 偏“全战系”的军事史诗感，容易压过你们的朝廷/门阀/案牍主题
- 大量战术表现层 UI

### 对《晚唐风云》的启发

它最有价值的是告诉你们：

- 中国题材策略 UI 可以是有力量感、有秩序感的
- 不必走“仙侠古风”或“手游国风”路线

### 适用模块

- 军事面板
- 战争概览
- 兵力与军团条目
- 势力入口与国家感表达
- 军政相关图标风格

---

## 3.3 Old World

### 适合参考的部分

- 高密度策略 UI 的整洁性
- 顶部资源信息排布
- 地图和面板的阅读节奏
- “复杂但不乱”的版式控制
- 信息优先级和视觉主次

### 不适合直接照搬的部分

- 西方古典题材的美术母题
- 过于理性、过于现代策略盘面的气质

### 对《晚唐风云》的启发

如果 CK3 是地图权力结构参考，那么 Old World 更适合当：

- UI 执行方法参考
- 可读性参考
- 信息密度控制参考

它能帮助你们解决的问题不是“风格像不像中国”，而是：

- 怎么在复杂系统里保持清晰

### 适用模块

- ResourceBar
- 各类数据列表
- 表格类面板
- tooltip
- 战争、经济、官职等数值密集模块

---

## 4. 辅参考游戏

## 4.1 Pentiment

### 适合参考的部分

- 材质语言统一
- 界面与时代背景的一致性
- 纸张、墨迹、边框、文字的整体协调
- “不是贴花，而是整套界面属于这个时代”的处理方式

### 不适合直接照搬的部分

- 过强的插画书/手抄本叙事风格
- 过度艺术化的文本与人物表现
- 低功能密度界面结构

### 对《晚唐风云》的启发

Pentiment 不适合作为系统 UI 结构参考，但非常适合做：

- 材质参考
- 文书感参考
- 历史氛围参考

特别适合启发你们的：

- 事件弹窗
- 史书界面
- 案牍式面板

### 适用模块

- ChroniclePanel
- EventModal
- 历史叙事相关弹窗
- 面板装饰语言
- 背景与纹理系统

---

## 4.2 Oriental Empires

### 适合参考的部分

- 中国大地图的空间气质
- 山川、水系、边疆地带的整体观感
- 中国地理的宏观分区感

### 不适合直接照搬的部分

- UI 完成度本身并不是最强项
- 地图与界面细节不一定足够现代和精致

### 对《晚唐风云》的启发

它不是最好的 UI 参考，但可以帮助你们把握：

- 什么叫“中国历史大地图”的呼吸感
- 什么叫“山川决定政区”的视觉直觉

### 适用模块

- 地图底图
- 山脉、水系、平原、边疆区域氛围
- 区域大轮廓和地理层次

---

## 5. 推荐的“学什么 / 不学什么”

为了避免参考失控，建议直接按以下原则执行。

### 5.1 可以重点学习

- 学 CK3 的地图层级与权力结构表达
- 学 CK3 的地图模式和边界优先级
- 学全面战争：三国的中国权力气质和军政氛围
- 学 Old World 的信息密度控制
- 学 Pentiment 的材质一致性
- 学 Oriental Empires 的中国地理氛围

### 5.2 不建议直接照搬

- 不建议照搬 CK3 的欧式卷轴、纹章、贵族装饰
- 不建议照搬三国全面战争的强写实军事演出感
- 不建议照搬 Old World 的西方古典风格
- 不建议照搬 Pentiment 的手抄本插画化视觉
- 不建议照搬 Oriental Empires 的整体 UI 工程细节

### 5.3 最终要转译成《晚唐风云》自己的语言

应转译出的关键词：

- 晚唐
- 节镇
- 朝廷
- 诏令
- 兵部
- 案牍
- 藩镇割据
- 舆图
- 边关

换句话说，参考项目提供的是“方法”和“构图思路”，最终气质必须仍然是《晚唐风云》自己的。

## 8. 建议的参考整理方式

建议团队后续把参考按模块分类，而不是按游戏名堆在一起。

推荐分类方式：

- 地图参考
- 资源栏参考
- 侧边菜单参考
- 人物面板参考
- 弹窗与事件参考
- 图标参考
- 字体与材质参考

这样做的好处是：

- 后续设计与开发更容易直接调用
- 不会陷入“我们到底是参考哪个游戏”的无效讨论

---

## 9. 结论

《晚唐风云》最适合采用“组合参考”策略，而不是寻找某一个可以完整照抄的对象。

推荐结论如下：

- 地图机制与地图层次：`Crusader Kings III`
- 中国历史权力气质：`Total War: THREE KINGDOMS`
- UI 可读性与信息结构：`Old World`
- 材质、文书感与时代一致性：`Pentiment`
- 中国大地形空间氛围：`Oriental Empires`

最终目标不是做成“像某个游戏”，而是提炼出一套属于《晚唐风云》自己的美术语言。
