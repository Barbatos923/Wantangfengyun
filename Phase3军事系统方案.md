# Phase 3 军事系统最终设计方案

> 基于多轮迭代讨论最终确定，2026-03-26

---

## 一、军队来源与编制

### 1.1 兵役人口

每个州根据人口数设定**兵役人口数**，即该州能招募的士兵总量上限。军事型领地 1户=0.2人（即20%户出兵），民政型领地 1户=0.05人（即5%户出兵）

### 1.2 两级编制：营 → 军

- **营**：军队最小实体单位，固定编制 **1000 人**
- **军**：由若干个营组成，是玩家主要管理的军队单位

玩家在州内进行**营的招募**（消耗兵役人口、降低发展度），招募后编入指定的军中。

---

## 二、营级单位属性

| 属性 | 说明 |
|:---|:---|
| 编制人数 | 固定 1000 人 |
| 兵种 | 重步兵、轻步兵、重骑兵、轻骑兵、弓箭手等（见兵种属性表） |
| 籍贯 | 招募所在州，如"太原"。影响士气和出征惩罚 |
| 驻扎地 | 当前所在州 |
| 士气 | 对当前统治者的忠诚度/士气值 |
| 精锐度 | 初始 0，训练可达 50，实战胜利可突破至 100，战败降低 |

### 兵种多维属性表

每个兵种拥有以下属性维度，不同阶段按权重生效，兵种优劣从数学中自然涌现，无硬克制：

| 属性 | 说明 |
|:---|:---|
| 冲击 | 正面冲锋能力 |
| 攻坚 | 对阵地/防御的突破力 |
| 追击 | 追歼溃敌能力 |
| 攻城 | 攻城战效率 |
| 行军速度 | 影响军级行军速度 |

> 具体兵种数值待实现时细化。

## 三、军级单位属性

| 属性 | 说明 |
|:---|:---|
| 规模 | 下辖营的总人数之和 |
| 兵马使 | 该军主将，影响军事力量层的叠加修正 |
| 士气 | 由各营士气加权平均得到 |
| 精锐度 | 由各营精锐度平均得到 |
| 驻扎地 | 当前所在州 |
| 行军速度 | 由各营行军速度加权平均得到 |
| 月粮耗 | 基础粮耗 = 总兵力 × 2 斛/月 |
| 月钱耗 | 待定 |
| 补给状态 | 是否充足，受补给线影响 |

## 四、士气系统

### 4.1 士气变化

| 触发条件 | 效果 |
|:---|:---|
| 赏赐钱财 | 士气增加 |
| 出籍贯地之外作战 | 士气持续缓慢下降（模拟牙兵出道战力下降、庞勋戍边叛乱） |
| 拖欠赏赐 | 士气下降 |
| 作战胜利 | 士气增加 |
| 作战失败 | 士气下降 |

### 4.2 士气影响

| 士气水平 | 效果 |
|:---|:---|
| 高昂 | 战斗中获得加成 |
| 低落 | 战斗中获得减益 |
| 极低 | 士兵背叛节度使：无法行动、临阵倒戈、杀死兵马使、节度使 |

## 五、战争理由（Casus Belli）

战争不能随意发动，需要**战争理由**。核心原则：合法性作为基础，消耗名望，可能花费金钱，并造成合法性变化。

### 5.1 武力兼并

任何情况下（除非四级集权）均可向相邻统治者使用：

| 时代 | 名望消耗 | 合法性损失 | 备注 |
|:---|:---|:---|:---|
| 治世 | 巨量 | 大量 | 代价极高 |
| 危世 | 大量 | 受损 | 代价较高 |
| 乱世 | 减少 | 无 | 转变为"逐鹿天下" |

### 5.2 宣称类

- **个人宣称**：通过伪造宣称、家族继承或皇帝命令获得
- **法理宣称**：拥有高级头衔（如节度使），自动拥有向占领该法理下属领地的其他藩镇宣战的理由
- **为他人宣称**：利用宫廷/幕府中他人的个人宣称对外宣战。胜利后该廷臣获得土地；若土地级别低于我方主头衔，其成为下属

### 5.3 时代特殊理由

- **危世**：奉召讨逆、矫诏征伐、独立
- **治世**：开疆拓土

## 六、行营与行军

### 6.1 行营

宣战后可组建**行营**。行营是地图上显示的图标实体，可聚集多个军的军队。

**行营都统**：默认由组成行营的军的将领中军事能力最高者担任，也可手动委派。行营都统影响**战术层**（策略抽取和战术力量计算）。

### 6.2 集结时间

| 军队所在位置 | 集结时间 |
|:---|:---|
| 本道内 | 当回合立即完成，可立刻行军 |
| 本道之外、本国之内 | 1个回合 |
| 本国之外 | 2个回合 |

### 6.3 行军计划

集结完毕后，每回合制定行军计划：根据行军速度选定目的地和行军路线。回合开始后军队逐天推进。

### 6.4 关隘与 ZOC

- 州与州之间非两两互通，有明确的**连线（路径）**
- 关键地理瓶颈（潼关、虎牢关、剑门关等）设有**关隘属性**
- 带有关隘属性的州产生 **ZOC（控制区域）**
- 军队**绝对无法**跨越未占领的 ZOC 节点直接攻击后方
- 想从长安打洛阳，必须先攻克潼关和陕州

### 6.5 防守方响应

被宣战后，防守方没有自动州兵/城防概念。必须：
1. 立即在重要关隘处组建行营
2. 聚集周边军队
3. 选择依城而守或出城野战

## 七、补给系统

### 7.1 补给线

军队在地图上移动时，每进入一个新州需要建立补给线。

```
月粮耗 = 基础粮耗 × 补给系数
补给系数 = 1 + (陆路距离 × 0.1) + (水路距离 × 0.03)
```

- 距离以"经过的州数"为单位
- 陆路每多经一州 +10% 消耗
- 水路每多经一州仅 +3% 消耗（水道利于补给）
- 占领途经州后，补给效率提高（该段距离/2）

### 7.2 补给不足

补给不足时，军队将产生额外损耗和士气下降。

## 八、攻城与围城

### 8.1 非关隘地区

占领速度较快。根据以下因素计算占领时间：
- 州的户数（规模）
- 围攻兵力数量

花费一定时间即可完成占领。

### 8.2 关隘地区

防守方（需有驻军）可选择：
- **野战**：出城迎击，进入标准四阶段战斗
- **守城**：双方同处关隘所在州，进入围城状态

围城为简单的时间计算模型，**不设阶段和事件**：
- 由敌我双方的攻城属性、数量、关隘等级综合计算
- 关隘等级越高，可支撑时间越长
- 守城方若不想坐以待毙，需出城野战
- **城破时，守城方所有士兵视为全部阵亡**

## 九、战斗系统

### 9.1 触发条件

双方军队交汇在同一个州时，触发战斗。

### 9.2 战斗阶段

每场战斗由四个阶段组成：

| 阶段 | 说明 | 阶段伤亡系数 |
|:---|:---|:---|
| 列阵 | 布阵试探，伤亡最轻 | 低 |
| 交锋 | 正面接触，伤亡中等 | 中 |
| 决胜 | 主力激战，互有伤亡 | 高 |
| 追击 | 战事已定，追杀敌军 | 高 |

- 前三阶段（列阵、交锋、决胜）均需评估胜负
- **决胜阶段胜者 = 追击阶段胜者 = 战斗胜者**
- 前三阶段双方互相造成伤害
- 追击阶段仅胜者对败者造成伤害

### 9.3 阶段胜负判定

由双方最终力量对比决定：

| 力量比 | 结果 |
|:---|:---|
| ≥ 2.0 | 完胜 |
| 1.5 ~ 2.0 | 大胜 |
| 1.0 ~ 1.5 | 小胜 |

## 十、战斗结算：分层模型

```
第一层：策略对撞 → 战术力量（纯指挥博弈）
第二层：军队对撞 → 军事力量（纯军队硬实力）
第三层：合并   → 阶段胜负 + 伤亡
```

### 10.1 第一层 — 战术力量

#### 策略选择流程

1. 每阶段基于行营都统的八维人格加权随机抽取候选策略
2. **玩家作为都统**：从加权随机的 3 条中手动选 1
3. **AI 战斗**：自动选择，后台快速结算

#### 策略定义

每条策略包含：
- **策略基础力量**
- **人格维度权重**：绑定八维人格，用于加权随机抽取
- **能力依赖**：影响战术力量计算
- **阶段适配**：某些策略不出现在特定阶段（如"穷追不舍"不出现在列阵阶段）
- **兵种适配**：某些策略需要特定兵种达到一定数量才可触发（如"铁骑冲锋"需要足够重骑兵）

#### 战术力量计算

```
战术力量 = 策略基础力量 × 能力系数 × 累计势头
```

- **能力系数**：都统对应能力维度的加成。能力低时可能 < 1
- **累计势头**：连续取得优势的一方获得递增势头加成
- 最终值在 **0 ~ 2** 之间，围绕 1 波动

#### 策略规模

- 按八维人格组织，每个人格维度在每个阶段**至少 3 条**候选策略
- 初期可以先做核心策略集，逐步扩充
- 追击阶段策略较简单（见 10.4）

### 10.2 第二层 — 军事力量

```
军事力量 = 军队数量 × 阶段属性加权值 × 叠加修正
```

#### 阶段属性权重

| 阶段 | 冲击 | 攻坚 | 追击 | 攻城 |
|:---|:---|:---|:---|:---|
| 列阵 | 0.2 | 0.8 | 0.0 | 0.0 |
| 交锋 | 0.8 | 0.1 | 0.1 | 0.0 |
| 决胜 | 0.4 | 0.6 | 0.0 | 0.0 |
| 追击 | 0.1 | 0.0 | 0.9 | 0.0 |

#### 叠加修正

- 精锐度加成
- 士气系数
- 兵马使（军级将领）军事能力加成
- 兵马使特质加成

### 10.3 第三层 — 合并

战术力量是**放大器**，不独立决定胜负：

```
甲方最终力量 = 甲方军事力量 × 甲方战术力量
乙方最终力量 = 乙方军事力量 × 乙方战术力量
```

最终力量大者取得该阶段胜利。

### 10.4 追击阶段特殊处理

追击阶段策略简化为固定选项：

**胜者（攻击方）**：
- 纵兵追杀 — 高伤害
- 穷寇勿追 — 低伤害，己方损失更少

**败者（防守方）**：
- 拼命逃窜 — 标准撤退
- 反戈一击 — 有机会反造伤害，但失败则损失更大

追击阶段**仅胜者对败者造成伤害**（除非败者选择"反戈一击"成功）。伤亡系数较高，追击属性为主要决定因素。

### 10.5 伤害计算

前三阶段，双方均对对方造成伤害：

```
部队减员 = 优势方最终力量 × 阶段伤亡系数 × 胜负关系修正
```

追击阶段仅胜者对败者造成伤害，伤亡系数较高。

## 十一、战后处理

### 11.1 战争分数

关键战役获胜及围城攻克获得战争分数。

#### 战斗产出的战争分数

```
敌方损失率 = 敌方总伤亡 / min(敌方总兵力, 50000)
```

| 等级 | 损失率 | 战争分数 |
|:---|:---|:---|
| 大胜 | ≥ 50% | +25 |
| 胜利 | ≥ 30% | +15 |
| 小胜 | ≥ 15% | +8 |

围城攻克另加分数（待定）。

#### 战争终结

| 条件 | 结果 |
|:---|:---|
| 攻击方战争分数达到 100% | 可要求对方**强制投降** |
| 任一方战争分数达到 50% | 可提出**和谈** |
| 攻击方久攻不下 | 防守方每月自动 +2 战争分数，至 50 后可提出和谈 |

### 11.2 减员补充

- **行营中不自动补员**
- 战胜方可获得战败方伤亡人数的**一定比例**作为俘虏补充
- 可对营进行**手动整编合并**（多个残营合为一营）
- 战后回到籍贯地进行招募补充，走征兵同样逻辑

## 十三、核心设计理念总结

1. **营是原子，军是管理单位，行营是战役单位** — 三级清晰分离
2. **籍贯+士气联动** — 复刻牙兵骄横、出道战力下降的历史特征
3. **战术层是放大器而非决定器** — 名将能放大优势，但不能以少胜十倍之敌
4. **ZOC+关隘** — 地缘铁律，无法偷家，必须正面攻坚
5. **无州兵概念** — 防守必须调动实际军队，后方空虚就是空虚
6. **战争分数制** — 明确的战争节奏控制，防止无限拖延
