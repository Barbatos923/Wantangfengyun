📖 项目简介

《晚唐风云》是一款深度聚焦于中国晚唐时期至宋元的角色扮演历史模拟游戏。玩家将扮演一位身处乱世的历史人物（如节度使、朝廷官员或游侠），在"藩镇割据"、"牙兵逐帅"、"宦官专权"的复杂政治生态中求生与扩张。

本项目不追求简单的地图涂色，而是致力于还原真实的中国式权力博弈。游戏最大的特色在于内置了AI 史书生成管线：玩家在游戏中的每一次决策、每一场战争、每一次人事变动，都会被系统记录为起居注，并在年度结算时，由大语言模型（LLM）自动编纂成一篇符合中国古代史书体例（纪传体/编年体）的文言文传记。

✨ 核心卖点 (USP)

1.
真实的中国式权力博弈：摒弃欧洲封建制模型，重构符合晚唐历史的系统。包括钱粮分离的经济系统、实职与散阶并行的官职系统、以及朝廷-藩镇-牙兵三方博弈的动态平衡。

2.
AI 自动生成专属史书：基于大语言模型的三层数据管线（事件捕获 → 起居注生成 → 史书编纂），将枯燥的游戏数值变动转化为"百战之将，威震殊俗"或"轻徭薄赋，百姓安居"的优美文言文史书，提供独一无二的"青史留名"成就感。

3.
天朝兴衰的宏观循环：引入"治世-危世-乱世"的全局状态机，模拟中国历史的王朝周期律。不同时代下，游戏的底层规则将发生根本性改变。


⚙️ 核心系统架构

本项目在设计上进行了深度的"中国化重构"，核心系统包括：

•
四维核心资源：钱（政治资本）、粮（军事资本）、名望（社会影响力）、合法性（统治权威）。特别设计了乱世中钱粮不可等价兑换的动态汇率机制。

•
双轨官职体系：官职分为实职（如节度使、刺史）与散阶（如光禄大夫），直接影响角色的合法性与俸禄。

•
牙兵与军事系统：将"牙兵忠诚"作为隐性核心指标并入军事系统。牙兵战斗力极强但极易哗变，完美模拟晚唐"长安天子，魏府牙军"的黑色幽默。

•
朝堂派系博弈：设计了尊王派、攘夷派、进取派、宦官集团、清流派五大派系，通过弹劾、推举等政治计谋进行互动。

•
三级地图架构：基于真实历史地理，构建三级 SVG 交互地图。




🛠️ 技术栈与 AI 管线

游戏前端与地图

•
地图渲染：基于 SVG 的交互式 Web 地图（支持缩放、拖拽、多图层切换）。

•
UI 框架：采用现代 Web 技术栈（如 React/Vue）构建复杂的系统面板与信息流。

AI 史书生成管线 (Chronicle Pipeline)

本项目最核心的技术挑战是实现高质量的文言文生成。我们设计了三层处理架构：

1.
事件记录层：游戏引擎实时捕获玩家行为，生成结构化的 GameEvent JSON 数据（包含时间、人物、地点、结果描述、重要性等级）。

2.
年度筛选层：每年腊月，系统根据重要性等级对全年事件进行过滤和上下文组装。

3.
AI 转化层：通过 ChronicleProvider 接口调用 LLM（支持 Kimi、Claude 等模型），结合精心调优的 Prompt，将结构化事件转化为文言文史书，并持久化存储至本地（IndexedDB/SQLite）。
