## 二、经济循环

### 4. 月收支计算
- [ ] 推进1个月，观察钱/粮变化合理
- [ ] 收入 = 领地产出 + 职位俸禄 + 属下上缴
- [ ] 支出 = 属下俸禄 + 上缴领主
- [ ] 上缴比例对应集权等级（1级25%, 2级45%, 3级65%, 4级85%）

### 5. ResourceBar月净值
- [ ] 顶部资源栏钱/粮旁显示月净值（绿色正值/红色负值）
- [ ] 鼠标悬停钱/粮图标，tooltip显示收支明细

### 6. 破产检查
- [ ] 钱或粮低于-500时：压力+10，所有属下好感-5
- [ ] （极端场景，较难自然触发）
